# GuideOS UI Guidelines

Version: **1.0**

Status: **Frozen**

---

# Purpose

Define the complete visual language for every GuideOS application.

These guidelines apply to

* Viewer
* Administration
* Future modules

Every screen shall appear as part of a single product.

No module may define its own visual language.

---

# Design Philosophy

GuideOS is an industrial engineering application.

The interface shall communicate

* Precision
* Reliability
* Clarity
* Simplicity
* Professionalism

Avoid decorative elements.

Every visual component shall have a functional purpose.

---

# Visual Principles

Prioritize

* Readability
* Consistency
* Predictability
* Low cognitive load
* Fast navigation
* Touch-friendly controls

---

# Color System

Every color shall use hexadecimal notation.

Example

```text
#0054A6
```

Never use

```text
white
black
red
green
blue
rgb(...)
```
